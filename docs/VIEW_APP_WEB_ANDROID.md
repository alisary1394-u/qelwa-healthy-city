# كيفية مشاهدة التطبيق على الويب أو أندرويد

## 1. تشغيل التطبيق على جهازك (ويب)

### المتطلبات
- Node.js مثبت ([nodejs.org](https://nodejs.org))
- ملف بيئة: انسخ `.env.example` إلى `.env.local` وضع قيم Base44 من لوحة التحكم

### الخطوات

```bash
# من مجلد المشروع
npm install
npm run dev
```

افتح المتصفح على: **http://localhost:5173**

ستظهر صفحة الترحيب وتسجيل الدخول (المدينة الصحية - محافظة قلوة). إذا لم تكن ربطت التطبيق بـ Base44 أو لم تضبط `.env.local` قد تظهر رسالة خطأ أو تحميل طويل — ضبط البيئة ضروري للاتصال بالخادم.

---

## 2. مشاهدة التطبيق على الجوال (أندرويد)

التطبيق **ويب** (React). يمكنك مشاهدته على أندرويد بإحدى الطريقتين:

### أ) من نفس الشبكة (Wi‑Fi) — للتجربة السريعة

1. على جهاز الكمبيوتر شغّل التطبيق مع فتحه لشبكتك المحلية:
   ```bash
   npm run dev -- --host
   ```
2. ستظهر في الطرفية عنوان مثل: `http://192.168.x.x:5173`
3. على أندرويد اتصل بنفس شبكة الـ Wi‑Fi وافتح **Chrome** وادخل نفس العنوان، مثلاً:  
   `http://192.168.x.x:5173`

بهذا تشاهد نفس واجهة التطبيق على الجوال.

### ب) بعد النشر على الإنترنت

عندما تنشر التطبيق على عنوان عام (مثل نشر Base44 أو أي استضافة):

1. ادخل الرابط العام من متصفح أندرويد (Chrome أو أي متصفح).
2. يمكنك إضافة الصفحة إلى الشاشة الرئيسية (قائمة المتصفح → "إضافة إلى الشاشة الرئيسية") لفتحه مثل تطبيق.

---

## 3. بناء نسخة للإنتاج (للموقع أو للمشاركة)

```bash
npm run build
```

الملفات الجاهزة للنشر ستكون في مجلد **`dist/`**. يمكنك رفع محتويات `dist/` إلى أي استضافة ثابتة (Netlify, Vercel, GitHub Pages، أو خادمك). إذا كنت تستخدم نشر الموقع من Base44، اتبع خطوات النشر من لوحة Base44 بعد ربط المستودع.

---

## 4. معاينة نسخة الإنتاج محلياً

بعد التنفيذ:

```bash
npm run build
npm run preview
```

ثم افتح الرابط الذي يظهر (عادةً http://localhost:4173) في المتصفح أو من الجوال على نفس الشبكة باستخدام IP جهازك بدل `localhost`.

---

## ملخص سريع

| الهدف | الأمر / الإجراء |
|--------|------------------|
| تشغيل على الكمبيوتر | `npm run dev` ثم افتح http://localhost:5173 |
| تشغيل + فتح من الجوال على نفس الـ Wi‑Fi | `npm run dev -- --host` ثم افتح http://192.168.x.x:5173 من الجوال |
| بناء للإنتاج | `npm run build` |
| معاينة نسخة الإنتاج | `npm run build` ثم `npm run preview` |
| أندرويد بعد النشر | افتح رابط التطبيق المنشور من متصفح الجوال |

إذا واجهت مشكلة في الطرفية (مثلاً بسبب مسار المشروع)، جرّب فتح **Command Prompt** أو **Git Bash** من مجلد المشروع وتنفيذ الأوامر من هناك.
