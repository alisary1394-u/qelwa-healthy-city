# دليل المستندات المطلوبة للمحاور (مرجع الفريق)

هذا الدليل يوفر قائمة مستندات مقترحة جاهزة كمرجع موحد للفريق عند تجهيز أدلة المعايير.

## قاعدة التسمية الموحدة

استخدم هذا النمط لجميع الملفات:

`[AXIS]-[STANDARD]-[DOC_TYPE]-[YYYY-MM-DD]-v[VERSION].[ext]`

**مثال:**
`A1-M1-3-MOM-2026-02-28-v1.pdf`

## المستندات الأساسية (تكرر مع كل معيار)

1. خطة تنفيذ المعيار
2. تقرير تقدم شهري
3. قائمة تحقق الامتثال
4. محضر اجتماع المتابعة
5. سجل المخاطر والإجراءات التصحيحية
6. دليل إثبات (صورة/تقرير/مستند)

---

## المحور 1: القيادة والحوكمة

- قرار تشكيل فريق المحور
- مصفوفة الأدوار والمسؤوليات
- خطة الحوكمة والمتابعة
- محاضر الاجتماعات الدورية
- تقرير الالتزام بالقرارات
- سجل المخاطر الإدارية

## المحور 2: التخطيط والسياسات

- الخطة التشغيلية للمحور
- خطة التنفيذ الزمنية
- السياسات والإجراءات المعتمدة
- نموذج متابعة تنفيذ الخطة
- تقرير الانحرافات والإجراءات التصحيحية
- دليل اعتماد التحديثات

## المحور 3: البيئة والصحة العامة

- خطة التفتيش البيئي
- تقارير جودة البيئة (هواء/مياه/نظافة)
- سجل الحملات الميدانية
- تقارير معالجة الملاحظات
- صور قبل/بعد التحسينات
- تقرير مؤشرات الصحة البيئية

## المحور 4: تعزيز الصحة والتوعية

- خطة التوعية الصحية
- المادة التوعوية المعتمدة
- جدول الأنشطة والفعاليات
- تقارير الحضور والاستفادة
- استبيان قياس الأثر
- تقرير التغطية الإعلامية

## المحور 5: المشاركة المجتمعية

- خطة إشراك المجتمع
- سجل الشراكات المجتمعية
- محاضر اللقاءات المجتمعية
- تقرير مبادرات التطوع
- سجل المقترحات والشكاوى
- تقرير الاستجابة المجتمعية

## المحور 6: الخدمات الصحية

- خطة تحسين الخدمة
- خرائط تدفق الخدمة
- تقارير الأداء التشغيلي
- تقرير زمن الانتظار
- سجل رضا المستفيدين
- خطة التحسين المستمر

## المحور 7: السلامة وإدارة المخاطر

- سجل المخاطر التفصيلي
- خطة الطوارئ والاستجابة
- تقارير اختبارات الجاهزية
- سجل الحوادث والإبلاغ
- تقرير تحليل السبب الجذري
- خطة الإجراءات الوقائية

## المحور 8: التحول الرقمي والبيانات

- سياسة حوكمة البيانات
- قاموس البيانات وتعريف المؤشرات
- تقارير جودة البيانات
- دليل التكامل بين الأنظمة
- تقارير لوحات المتابعة
- خطة حماية البيانات والنسخ الاحتياطي

## المحور 9: الاستدامة والتحسين المستمر

- خطة الاستدامة
- مؤشرات قياس الأثر طويل المدى
- تقرير مراجعة الأداء السنوي
- خطة التحسين السنوية
- تقرير الدروس المستفادة
- خطة نقل المعرفة للفريق

---

## جدول اختصارات أنواع المستندات (للاسم الموحد)

- `PLAN` خطة
- `PROGRESS` تقرير تقدم
- `CHECKLIST` قائمة تحقق
- `MOM` محضر اجتماع
- `RISK` سجل مخاطر
- `EVIDENCE` دليل إثبات
- `AUDIT` تقرير مراجعة
- `CAPA` إجراءات تصحيحية/وقائية

## مثال جاهز لكل معيار

1. `A3-M3-2-PLAN-2026-01-15-v1.docx`
2. `A3-M3-2-PROGRESS-2026-02-28-v1.docx`
3. `A3-M3-2-CHECKLIST-2026-02-28-v1.xlsx`
4. `A3-M3-2-MOM-2026-02-20-v1.pdf`
5. `A3-M3-2-RISK-2026-02-28-v1.xlsx`
6. `A3-M3-2-EVIDENCE-2026-02-28-v1.pdf`
